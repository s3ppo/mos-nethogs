# Third-Party Software

This plugin packages and relies on the following open-source software:

- **Nethogs** - Network traffic monitor per process
  - License: GPL-2.0
  - Source: https://github.com/netoptimizer/nethogs
  
- **iftop** - Interactive network bandwidth monitor (optional)
  - License: GPL-2.0
  - Source: http://www.ex-parrot.com/~pdw/iftop/
  
- **vnstat** - Network traffic statistics tool (optional)
  - License: GPL-2.0
  - Source: http://humdi.net/vnstat/
  
- **bmon** - Bandwidth Monitor (optional)
  - License: MIT
  - Source: https://github.com/tgraf/bmon

This plugin bundles configuration and installation scripts for MOS.
All third-party components remain licensed under their original upstream licenses.
